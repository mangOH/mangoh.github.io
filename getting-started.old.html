Getting started

Go ahead, build something new!
To get up and running quickly on your first mangOH™ Green board,
feel free to follow these discovery tutorials and step-by-step instructions.

 Setup your Hardware
_


1 Note the serial number (FSN) and IMEI of your CF3 module (used later to identify your device)
2 Mount the CF3 module in the primary socket
3 Connect main RF antenna



4 Connect GNSS antenna
5 Insert mini or micro SIM (without PIN code)
6 Connect power select jumper and power supply



For more info, read the   User Guide

Now you’re ready for the next step!

 Develop your first HelloWorld
_

mangOH™ Green is using the Legato open source embedded platform built on Linux to design the software of an IoT application.
Here are the steps you’ll need to take to compile and load your first Legato HelloWorld on your target for:
  • ◆   Command Lines Users with Linux PC Host
  • ◆   Graphical Interface Users with Windows PC Host : follow the installation wizard and HelloWorld cheat sheet
  • ◆   Graphical Interface Users with Linux PC Host : follow the installation wizard and HelloWorld cheat sheet


Congratulations! Now that you wrote your first HelloWorld, we invite you to connect your mangOH Green to the cloud (see section below). For more detailed documentation and samples, go to    legato.io


 Connect to the Cloud
_

mangOH™ Green is natively connected to AirVantage allowing you to send applicative data and receive commands. Start these 2 tutorials from   here:

  • ◆   Get Ready to Develop Legato Application
  • ◆   Develop your application to communicate with AirVantage


For more detailed documentation, go to the     AirVantage documentation
