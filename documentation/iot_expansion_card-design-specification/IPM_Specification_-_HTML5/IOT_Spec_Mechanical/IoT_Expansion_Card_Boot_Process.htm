4.4 IoT Expansion Card Boot Process

When an IoT Expansion Card is detected, the host application reads the expansion card’s EEPROM header information (see EEPROM Header.) This includes the expansion card’s slot width (number of slots it occupies), which the host needs to determine each expansion card’s location. For example, if a 2-slot wide expansion card is in slots 0–1, the host knows the next available slot is slot 2.
