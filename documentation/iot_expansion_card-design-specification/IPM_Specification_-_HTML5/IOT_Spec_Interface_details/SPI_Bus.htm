3.6 SPI Bus

The IoT Expansion Card supports a 3/4/5-wire serial peripheral interface (SPI) that may be used for data communication between the expansion card and a host application.

The following features are available on the SPI bus:

•Mode: Slave (Master mode is not supported)

•3/4/5-wire interface

Note:   Although the IoT Expansion Card supports 3, 4, and 5-wire implementations of the SPI bus, it will only work if the host application supports the same architecture (number of signals, bus speed, data length) as the expansion card.
