 CF3 modules
_

mangOH designs accepts the following CF3 modules & application processors. To run properly, mangOH designs need 1 application processor supporting Legato™, an open source Linux-based O/S and application framework. Legato provides useful libraries for applications running on mangOH™ platforms.

Distributors of mangOH boards referenced above are distributing the following mangOH compatible CF3 modules:

AirPrime® WP8
Wireless (3G+GPS) processor from Sierra Wireless running Legato , an open source Linux based O/S & framework for IoT
AirPrime® HL8
Wireless (3G+GPS) modem from Sierra Wireless that needs to be associated to an application processor on the other CF3 socket
Coming soon...
More CF3 modules and application processors coming soon, including 2G, 4G, LTE-M and non cellular too. Stay tuned!
